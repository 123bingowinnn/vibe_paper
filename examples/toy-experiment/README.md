# Toy Experiment Project

This example project imitates a small computer vision experiment repository.

It contains:

- source code in `src/`
- training configuration in `configs/`
- exported metrics in `results/`
- plain training logs in `logs/`

The purpose of this project is to demonstrate how Vibe Paper attaches to a real experiment folder, generates `paper/context/project_snapshot.md`, and allows an agent to draft a paper from actual project evidence rather than from an empty template.
