# Claude Code Adapter

Copy `CLAUDE.md.template` into the experiment project root as `CLAUDE.md`, then adjust any local wording you want.

The goal is simple:

- Claude Code should read the whole project
- `paper/context/project_snapshot.md` should be the first paper-writing context file
- `paper/` should remain the writable LaTeX workspace
- the local Vibe Paper runtime should remain the source of compile and preview behavior
