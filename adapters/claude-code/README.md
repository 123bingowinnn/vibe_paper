# Claude Code Adapter

Copy `CLAUDE.md.template` into the experiment project root as `CLAUDE.md`, then adjust any local wording you want. The canonical Vibe Paper instructions live in the root `SKILL.md`; this folder only keeps Claude Code-specific notes.

The goal is simple:

- Claude Code should read the whole project
- `paper/context/project_snapshot.md` should be the first paper-writing context file
- `paper/` should remain the writable LaTeX workspace
- the local Vibe Paper popup preview and compile runtime should remain the source of preview behavior
